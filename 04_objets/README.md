# Devoir : Objets

Pour ce devoir, vous devez concevoir un programme qui affiche le jour de
la semaine (lundi, mardi, etc.) auquel correspond une date saisie par
l'utilisateur·rice. En plus de la date, l'utilisateur·rice doit saisir
la langue (français ou anglais) dans laquelle le jour de la semaine est
affiché. Affichez un message d'erreur si la date ou la langue saisie est
invalide.

Avant de commencer, on vous conseille fortement de prendre quelques
minutes pour consulter la [documentation de Date].

[documentation de Date]: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date
