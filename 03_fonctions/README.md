# Devoir: Fonctions

Pour ce devoir, vous devez concevoir un programme qui calcule le prix
total d'une commande de chandails personnalisés. Votre programme doit
utiliser `prompt` pour demander à l'utilisateur·rice la quantité de
chandails qu'il ou elle veut acheter, et le message qui doit être
imprimé sur ceux-ci. Un chandail coûte 10$, plus 50¢ pour chaque
caractère du message (excluant les espaces). Le total doit inclure les
taxes fédérales (5 %) et provinciales (9,975 %). Utilisez `alert` pour
afficher le résultat.

Vous aurez besoin de la fonction [replaceAll].

[replaceAll]: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/replaceAll
